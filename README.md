PROMPT
======

MAXIMALIST ENTERPRISE WEBSITE PROMPT
=====================================

CONTEXT: Design a maximalist, enterprise-grade website for [INSERT YOUR BUSINESS DESCRIPTION]. Create a visually striking, professionally polished web experience that commands attention while maintaining credibility.

ROLE: Expert web designer and creative technologist specializing in shader-based animations, interactive visual effects, and cutting-edge web experiences. Blend maximalist aesthetics with enterprise-level execution.

---

CORE REQUIREMENTS
- Project Type: Enterprise-grade website
- Purpose: [INSERT BUSINESS PURPOSE]
- Design Approach: Maximalist aesthetic
- Scope: Minimum 4-5 complete sections

REQUIRED SECTIONS
1. HERO: Custom shader-based animated text, interactive mouse hover effects, beautiful typography. CRITICAL: Text must be fully responsive, never overflow viewport.
2-5. ADDITIONAL (pick 3-4): Services/Features | About/Company | Portfolio/Case Studies | Testimonials/Social Proof | CTA/Contact | Process/How It Works

VISUAL TREATMENT (all sections): Dithering textures, wavy/fluid motion, groovy animations, rich gradient overlays

STYLE KEYWORDS: maximalist • groovy • wavy • gradients • dithering • enterprise-grade

VISUAL CHECKLIST
[ ] Shader effects on hero text
[ ] Mouse hover-interactive animations
[ ] Gradient color schemes
[ ] Dithering texture layers
[ ] Wavy motion design

TECHNICAL REQUIREMENTS
- Enterprise-level polish
- Optimized performance despite heavy visuals
- Smooth shader rendering
- Responsive mouse tracking
- CRITICAL: Hero text uses responsive font sizing (clamp, vw units, or media queries)
- CRITICAL: Prevent horizontal overflow - all text fits within viewport
- Proper container constraints and text wrapping
- Test across mobile, tablet, desktop breakpoints

---

EXECUTION WORKFLOW
1. PLANNING: Define structure, plan shader implementation, map interaction flows, outline animations
2. DESIGN SYSTEM: Color palette/gradients, typography hierarchy, dithering/texture treatments, component variations
3. IMPLEMENTATION: Shader infrastructure, mouse tracking system, animation framework, performance optimization
4. DELIVERABLE: Complete single-file HTML with all CSS/JS inline, ready to copy/paste, fully interactive
